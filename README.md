# clash-clan-manager

Requires python3.9
<br>Add data/config.json

```
{
	"token": "xxxxxxx",
	"clanTag": "xxxxxxxx",
	"key": "Notes password",
	"email": "API Account email https://developer.clashofclans.com/",
	"password": "API Account password",
	"keyName": "clanManager",
	"history": 8
}
```

<br>Add data/warlog.json
```
{
  "items": []
}
```

```
pip3 install -r requirements.txt
run.sh
```
